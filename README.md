# Gemmi Homepage

Static website for the Gemmi Android app.

## Pages

- `index.html` - Landing page
- `privacy-policy.html` - Privacy Policy URL for Google Play
- `data-deletion.html` - Account and data deletion URL for Google Play
- `terms.html` - Terms of Service

## Deploy

You can deploy this folder on GitHub Pages, Vercel, Netlify, or Cloudflare Pages.

Recommended Google Play URLs after deployment:

- Privacy Policy: `https://your-domain/privacy-policy.html`
- Account deletion: `https://your-domain/data-deletion.html`
- Terms: `https://your-domain/terms.html`
